Essa é uma aplicacao de console. Para navegar entre suas funcionalidades, utilize um teclado numerico seguido do Enter.
Os dados serão lidos do arquivo "dados_urna", nele ja ha dados. Tambem e possivel importar dados csv.
Dados lidos do csv serao adicionados a base de dados "dados_urna".
Utilize o arquivo "dados_input" como exemplo de importacao de dados csv.
